## 📋 配置文件结构

脚本生成的完整配置结构：

```
~/emby2alist-config/
├── docker-compose.yml              # Docker 编排文件
├── .env.example                    # 环境变量模板
├── nginx.conf                      # Nginx 主配置
├── conf.d/
│   ├── constant.js                 # 核心配置（脚本生成）
│   ├── emby.conf                   # Emby 路由配置
│   ├── emby.js                     # 主逻辑脚本
│   └── config/
│       ├── constant-common.js      # 公共常量
│       ├── constant-mount.js       # Alist 配置（脚本生成）
│       ├── constant-pro.js         # 高级配置（脚本生成）
│       ├── constant-symlink.js     # 符号链接配置
│       ├── constant-strm.js        # STRM 文件配置
│       ├── constant-transcode.js   # 转码配置
│       ├── constant-ext.js         # 扩展功能配置
│       └── constant-nginx.js       # Nginx 特定配置
├── log/                            # 日志目录
└── embyCache/                      # 缓存目录
```
