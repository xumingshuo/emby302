## 📄 文件说明

### 1. setup-emby2alist.sh
**类型**: Bash 脚本  
**大小**: ~20KB  
**行数**: ~600 行  
**用途**: 自动化配置 emby2Alist

**主要功能**:
- 交互式收集配置信息
- 验证 Emby/Alist 连通性
- 自动计算 mediaMountPath
- 生成配置文件
- 创建 Docker Compose
- 启动服务

**使用方法**:
```bash
chmod +x setup-emby2alist.sh
./setup-emby2alist.sh
```
